MiniMax 开源 M2.7 模型，SWE-Pro 评测得分达 56.22%

MiniMax 公司宣布其 M2.7 模型已在 ModelScope 平台开源。据文中介绍，该模型在开发过程中能够帮助构建强化学习工具链、更新自身记忆并驱动自身的训练循环，从而成为迭代自身的工具。在软件工程能力评测中，其 SWE-Pro 得分达到 56.22%（文中称与 GPT-5.3-Codex 持平），Terminal Bench 2 得分为 57.0%。文中附带了指向 Office 评测结果的链接。

vLLM 推理框架在发布首日即支持 MiniMax M2.7 模型

MiniMax 官方宣布，其 M2.7 大语言模型在发布首日（Day-0）即获得 vLLM 推理框架的支持。根据官方文档，MiniMax-M2 系列模型（包括 M2.7、M2.5、M2.1 和 M2）在数学、科学、编码和工具使用方面位列全球开源模型榜首，并在高级编码任务上表现出色，具备强大的智能体性能。该模型采用高效设计，拥有 100 亿激活参数（总计 2300 亿）。部署要求包括 Linux 系统、Python 3.10-3.13，并详细说明了 NVIDIA GPU 和 AMD GPU 的多种推荐配置。单个序列最大上下文长度为 19.6 万令牌。部署方式支持 Docker 镜像、安装特定 ROCm wheel 或从源码安装特定版本。

MiniMax M2.7模型登陆NVIDIA GPU加速端点，支持NemoClaw与OpenClaw

MiniMax公司宣布，其M2.7模型现已在NVIDIA GPU加速端点上可用，并支持包括NemoClaw和OpenClaw在内的工具。M2.7是M2系列的最新版本，旨在增强智能体工作流及推理、机器学习研究等复杂应用。它是一个稀疏混合专家模型，总参数量为2300亿，每令牌激活参数量为100亿，激活率约4.3%，支持20万令牌的上下文长度，拥有256个本地专家，每令牌激活8个专家，共62层。为支持部署，NVIDIA推出了开源参考堆栈NemoClaw，以简化OpenClaw的安全部署，并可通过NVIDIA Brev云平台一键启动。此外，NVIDIA还与开源社区合作，在vLLM和SGLang等框架中集成了针对大规模MoE模型优化的高性能内核。

Gartner精选IT新闻：AI代理组建近全自主软件工程团队

Gartner近日分享了本月IT新闻精选，其中《华尔街日报》报道了硅谷初创公司JustPaid利用OpenClaw和Anthropic的Claude Code AI编码工具，组建了一支由7个AI代理组成的近乎全自主的软件工程团队。这些代理能够24/7不间断工作，在一个月内完成了10个原本各需人类开发者耗时一个月或更久才能完成的主要功能。OpenClaw作为开源的AI代理编排系统，充当决策大脑，而Claude Code则负责执行具体的编码任务。类似地，初创公司Kuse也使用OpenClaw构建了“AI同事”。然而，文中也指出此类自主代理存在风险，需要持续监督，可能篡改或删除文件，且成本可能因“令牌”使用量累积而变得高昂。大型企业如英伟达正在开发更安全的企业级工具。市场分析师认为，AI代理将接管人类工程师的大部分工作，导致开发团队规模缩小，未来人类工程师的角色可能转变为管理者。
